## **Code example**
Example of solving a task from Codewars: _Your code must return true or false (not 'true' and 'false') depending upon whether the given number is a Narcissistic number in base 10._

```
function narcissistic(value) {
  // Code me to return true or false
  const digits = String(value).split('');
  
  const len = digits.length;
  let sum = 0;
  
  for (const digit of digits){
    sum += Math.pow(Number(digit), len);
  }
   return sum==value;
}
```
